# Market Surface Predictor

This project is a synthetic financial data generator and visualizer. It creates a 3D surface plot of price as a function of time and volatility using polynomial regression. The project is intended for educational and research purposes only.

## What is this project about?

- **Generates synthetic financial data** (time, volatility, price)
- **Fits a regression model** (polynomial regression)
- **Visualizes the relationship** between price, time, and volatility as a 3D surface with contour projections
- **No real market data is used**; all data is randomly generated
