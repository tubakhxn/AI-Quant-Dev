# AI + Quant Development Projects

This repository contains three Python projects that combine AI, quantitative modeling, and data visualization using fully synthetic data. Each project is lightweight, self-contained, and built for educational purposes.

---

# 1. Market Surface Predictor

## What is this project about?

Generates synthetic financial data and visualizes price as a 3D surface over time and volatility using regression.

## How to run

pip install numpy matplotlib scikit-learn
python market_surface_predictor.py

## Relevant Topics

Financial markets
Volatility
Polynomial regression
3D surface plots

## Disclaimer

This project is for educational and research purposes only and does not provide financial predictions.

---

# 2. Smart Backtesting Engine

## What is this project about?

Simulates a trading strategy using synthetic stock data, ML-based signals, and visualizes performance metrics like equity curve and drawdown.

## How to run

pip install numpy pandas matplotlib
python smart_backtest.py

## Relevant Topics

Backtesting
Moving averages
Random walk
Drawdown analysis

## Disclaimer

This project is for educational and research purposes only and is not intended for real trading or financial advice.

---

# 3. Urban Data Modeling with AI

## What is this project about?

Models relationships between distance, population density, and housing prices using ML and visualizes them with smooth curves, heatmaps, and 3D plots.

## How to run

pip install numpy matplotlib seaborn scikit-learn scipy
python urban_data_modeling.py

## Relevant Topics

Urban modeling
Regression analysis
Population density
Housing data

## Disclaimer

This project is for educational and research purposes only and not for commercial use.
