# Urban Data Modeling with AI

This project is a standalone Python script for generating, modeling, and visualizing synthetic urban data. It demonstrates how AI and data science techniques can be used to explore relationships between urban features such as distance from city center, population density, and housing prices.

## What is this project about?

- **Synthetic Data Generation:** Automatically creates realistic urban data for analysis.
- **Regression Modeling:** Fits polynomial and random forest regression models to understand relationships between variables.
- **Visualization:** Displays scatter plots with regression curves, density heatmaps, and a 3D surface plot, all in a single window.
- **Educational Purpose:** Designed for learning, research, and demonstration of urban data modeling techniques.

## How to fork and run

1. **Fork this repository** (if on GitHub, click the "Fork" button at the top right).
2. **Clone your fork:**
   ```bash
   git clone https://github.com/your-username/urban-data-modeling-ai.git
   cd urban-data-modeling-ai
   ```
3. **Install requirements:**
   ```bash
   pip install numpy matplotlib seaborn scikit-learn scipy
   ```
4. **Run the script:**
   ```bash
   python urban_data_modeling.py
   ```
